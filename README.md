## üß≠ Overview
**Ifenkiri** is designed to help users explore a wide range of movies effortlessly.  
With intuitive search and filter options, a responsive design, and detailed movie information, it serves both casual viewers and devoted cinephiles.  
By integrating **Appwrite** and **TMDb**, the app ensures robust backend support and accurate, up-to-date movie data.

---

## ‚ú® Features
- **Movie Discovery:** Browse through a curated list of movies.  
- **Search & Filter:** Quickly find movies by title, genre, or release date.  
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices.  
- **Detailed Movie Information:** Access details such as synopsis, cast, ratings, and reviews.  
- **Modern Stack:** Built with React for dynamic UI and Tailwind CSS for efficient styling.  
- **Backend Integration:** Appwrite handles authentication, database, and file storage.  
- **Reliable Movie Data:** Powered by the TMDb API for rich and accurate information.
